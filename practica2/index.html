- Resumen -

En ésta práctica conocimos el funcionamiento de algunas etiquetas empleadas en la programación c++, sus parámetros y la forma correcta de como utilizarlas.

En esencia los códigos que aparecerán a continuación, tienen la misma estuctura, pero es gracias a las diferentes combinaciones de etiquetas y que son diferentes entre sí.

- Introducción -

"C++ es un lenguaje de programación de propósito general que extiende las capacidades de C al agregar nuevas características como la programación orientada a objetos. Desarrollado en 1979 por Bjarne Stroustrup, C++ fue diseñado para ofrecer un mayor control sobre los recursos del sistema, combinando la eficiencia de C con las ventajas de los lenguajes modernos.
C++ permite a los programadores crear aplicaciones que van desde software de sistemas hasta videojuegos y aplicaciones de alto rendimiento, gracias a su capacidad para gestionar memoria y su flexibilidad."

Robledano, A. (2019, 22 de julio)

En esta parte se abordarán los parámetros que tiene que tener la página y las etiquetas utilizadas para dar formato, con el fin de familiarizarnos con sus distintos usos.

- Etiquetas -

"< !DOCTYPE html >"

"< html > < /html >"

"< head > < /head >"

"< title > < /title >"

"< body > < /body >"

"< h1... > < /h1... >"

"< strong > < /strong > "

"< p > < /p >"

"< img >"

"< a > < /a >"



- Desarrollo -

Existen muchos programas de edición de código c++, en los que puede llegar a variar el formato o algunas funciones, pero las etiquetas son algo universal y que en contadas ocaciones llegan a variar.

Una vez teniendo en mente las etiquetas y dónde queremos llevar página que queremos programar, tenemos que guardar toda la información necesaria en una carpeta fácil de ubicar y, dentro de ella, crear un programa (.html) con el nombre de "index", ya que va a ser nuestra página principal.

Planeación

Una vez generado un index, sigue darle formato:

La etiqueta que le da vida a una página html "< !DOCTYPE html >", con la cual se nos activan las demás funciones, seguido de esto se activa < html > < /html > y todo lo que pongamos dentro de este dominio va a ser nuestra página, hay "grupos" asignados a lo que va a aser nuestra página, y que en esta ocación vendrían a ser: "< head > < /head >" y "< body > < /body >" .

Diseño

Dentro del dominio de head vamos a poner información referente al nombre de la página (como nombre, intitución, grupo, etc.), y dentro de body va a estar lo que podemos ver, como imágenes, videos, texto, etc.

Las etiquetas seleccionadas en esta ocación seran de diseño simple, como "< h1... > < /h1... >" las cuales tienenla función de poner títulos, subtitulos, etc.

Construcción

En nuestro index vamos a crear una página principal que aloje a nuestras extenciones o practicas que tengamos que poner, lo cual se logra a travez de hipervínculos que se dan con la etiqueta "< a > < /a >", con la cual se pueden hacer una infinidad de combinaciones para lograr formatos, combinar con imágenes o hacer lo que se necesite en el momento, a las cuales se les puede dar formato con "< strong > < /strong > " o "< b > < /b >", las cuales (por dar un ejemplo), las pone en negritas; y en esta ocasión, se utilizará para conectar nuestras páginas, para lo cual debemos crear una carpeta dentro de donde se encuentra el index y ahí hay que seguir los mismos pasos para crear una segunda página.



Se pueden incertar imágenes con "< img >" y a esta se le puede dar una serie de formatos que le darán mucha más presentación a nuestra página.

- Resultados -

Las páginas diseñadas esta vez fueron muy básicas, e incluian tener un título, un subtítulo, texto, imagenes e hipervínculos que conecten a páginas con el mismo concepto.

- Conclusiones -

Gracias a los diferentes editores de código c++, se pueden realizar una serie de combinaciones para darle formato a una página web, logrando desarrollar un programa desde 0 sin muchas complicaciones.

Estas páginas son capace de llevarte a otras páginas creadas por los mismos usuarios o a cualquiera que sea de dominio público.
