Módulo Bluethoot

Protoboard Electrónica

- Resumen -

La práctica se centró en el uso de Bluetooth para el control de dispositivos, empleando Arduino como herramienta principal. Se investigaron las funcionalidades del módulo Bluetooth y su conexión con Arduino, lo que permitió establecer un control remoto sobre diferentes dispositivos..

- Introducción -

Esta práctica se centra en el control de dispositivos mediante tecnología Bluetooth, utilizando Arduino como plataforma central.

A través de esta pracrica, se busca no solo adquirir habilidades técnicas en la programación y configuración de estos componentes, sino también comprender las amplias aplicaciones que surgen al combinar la potencia del Arduino con la versatilidad de la conectividad Bluetooth.

- Desarrollo -

El desarrollo del sistema a través de MIT App Inventor y Arduino fue un proceso metódico que combinó diseño, programación y pruebas prácticas. La integración de estos elementos resultó en una solución efectiva para el control remoto de dispositivos, destacando la versatilidad y el potencial de Bluetooth en aplicaciones de automatización.

Diseño

Se diseño la practica utilizando la plataforma MIT App Inventor, una herramienta intuitiva que permite el desarrollo de aplicaciones móviles de manera sencilla y accesible. A través de esta plataforma, se creó una aplicación específica destinada a controlar dispositivos conectados al Arduino mediante Bluetooth.

Se diseñó la interfaz de la aplicación, seleccionando los elementos visuales necesarios, como botones y sliders, para facilitar la interacción del usuario. Posteriormente, se desarrolló un código utilizando el sistema de bloques de MIT App Inventor, que permitió establecer la lógica de funcionamiento de la aplicación.

Construcción

Para que funcionara se utilizo un codigo, el cual hace que se prenda un disposito a través de Bluetooth.

#include /SoftwareSerial.h>

SoftwareSerial mySerial(2,3); //TX, RX

char Dato='X';

void setup()

{

mySerial.begin(9600); //Iniciamos Software Serial

Serial.begin(9600);

pinMode(13,OUTPUT);

}

void loop()

{

if(mySerial.available()>=1){ //Si recibimos datos

Dato=mySerial.read(); Serial.println(Dato);

if(Dato=='1'){

digitalWrite(13,HIGH);

}

else

{

digitalWrite(13,LOW);

}

}

}

- Resultados -

La actividad se evaluó de manera positiva, logrando todos los objetivos planteados, ya que se hizo que funcionara la aplicacion con las indicaciones que se mostraban con los botones.

- Conclusiones -

Esta práctica nos permitió comprender de manera más profunda las posibilidades que ofrece el control de dispositivos a través de Bluetooth. Al utilizar el módulo Bluetooth en combinación con Arduino, hemos observado cómo esta tecnología facilita la interacción y el manejo de distintos sistemas de manera remota.

A lo largo de la práctica, aprendimos no solo a establecer la conexión entre el Arduino y el módulo Bluetooth, sino también a explorar diversas aplicaciones que podemos implementar en proyectos futuros. Esto abre un abanico de oportunidades para el desarrollo de soluciones innovadoras y eficientes, que pueden ser aplicadas en múltiples áreas, desde la domótica hasta la robótica.
